-------------------
Описание программы:
-------------------

"Anemobarometer interface" - Программа для отслеживания и отображения скорости и направления ветра с анеморумбометра, 
производимого компанией Escort Monitoring Systems с функцией записи отчета в MS Excel.
Запись логов возможна только при запуске приложения от имени администратора.

--------------------------
Поддерживаемые устройства:
--------------------------

1. Сокол-А

---------------------
Требования к системе:
---------------------

1. Windows 10, версии 16299 (Fall Creators Update) или выше;
2. Python 3.11 или выше;
     2.1. Для установки Python последней версии перейдите по ссылке (при установке поставить флаг "Add python.exe to Path"): https://www.python.org/downloads/
     2.2  Инструкция для установки pip в переменные среды: https://setiwik.ru/kak-ustanovit-pip-dlya-upravleniya-paketami-python-v-windows/ 
4. Для работы программы необходимо постоянное Wi-Fi соединение.


-----------------
Запуск программы:
-----------------

1. Подключите Анеморумбометр Сокол-А и компьютер к одной WiFi-сети;
2. Откройте папку "main". Внутри папки запустите файл с именем "main" от имени администратора;
3. При первом запуске программы произойдет подгрузка необходимых библиотек. Это займет несколько минут;
4. После загрузки библиотек произойдет открытие окна DASH в браузере по умолчанию, на котором отображается 4 графика.


-------------------------
Дополнительные пояснения:
-------------------------

1. Графики
     1.1. В открытом окне браузера страница Dash, содержащая 4 графика. Первый - отображает реальные данные с устройства.
     1.2  2-4 графики получены с помощью генератора чисел и лишь демонстрируют визуальный функционал программы.
     В последствии графики 2-4 могут быть заменены на отображение реальных данных с устройств этой модели.

2. Запись логов
     2.1  Одновременно с отображением данных в виде графика "Розы ветров" происходит их запись в файл csv
     2.2  Имя файла: ГГГ-ММ-ДД.csv(сегодняшняя дата). Файл находится в папке logs внутри папки Anemobarometer interface. 
     Запись ведется только для первого (настоящего) графика.

4. Окно предупреждения
     4.1  В программе заложен функционал, позволяющий вывести окно предупреждения в случае превышения скорости ветра
     выше заданного порогового значения. 
     4.2  Сейчас окно загорается на 5 секунд при скорости ветра > 17,5 м/с.
     4.3. В окне предупреждения также выводится скорость ветра, спровоцировавшая отображение окна.
