Программа для отслеживания скорости и направления ветра с флюгера-анемометра, а также выгрузки логов полученных значений.
Запись логов возможна только при запуске приложения от имени администратора.

Запуск:
1. Откройте папку "main". Внутри папки нажмите два раза на файл с именем "main".
2. При первом запуске программы произойдет подгрузка необходимых библиотек. Это займет несколько минут.
3. После загрузки библиотек произойдет открытие окна DASH в браузере по умолчанию, на котором отображается 4 графика.


Пояснение:
1. В открытом окне браузера страница Dash, содержащая 4 графика. Первый - отображает реальные данные с устройства.

2. 2-4 графики получены с помощью генератора чисел и лишь демонстрируют визуальный функционал программы.
   В последствии графики 2-4 могут быть заменены на отображение реальных данных с устройств этой модели.

3. Одновременно с отображением данных в виде графика "Розы ветров" происходит их запись в файл csv, 
   который находится в папке logs внутри папки main. Запись ведется только для первого (настоящего) графика.

4. В программе заложен функционал, позволяющий вывести окно предупреждения в случае превышения скорости ветра
   выше заданного порогового значения. Сейчас окно загорается на 5 секунд при скорости ветра > 17,5 м/с.
   В окне предупреждения также выводится скорость ветра, спровоцировавшая отображение окна.
